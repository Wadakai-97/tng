# Laravelで始める場合
composer create-project laravel/laravel="X.X.*" firstname-lastname

# Svelteで始める場合
npm create svelte@latest firstname-lastname

## 依存パッケージのインストール
npm install

## アプリケーション起動
npm run dev

# solid.jsで始める場合
npx degit solidjs/templates/js firstname-lastname

## 依存パッケージのインストール
npm install

## アプリケーション起動
npm run dev
